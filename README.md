# Crepe Phahon Yothin 35

A bilingual (Thai/English) website for a crepe shop located in Phahon Yothin 35, Bangkok. The website features a custom crepe builder, location information, and a responsive design.

## Features

- 🥐 Custom Crepe Builder
- 🌐 Bilingual Support (Thai/English)
- 📍 Google Maps Integration
- 📱 Responsive Design
- 🎨 Interactive UI

## Technologies Used

- HTML5
- CSS3
- JavaScript/jQuery
- Bootstrap 5
- Google Maps API


## Language Support

The website automatically defaults to Thai language and supports switching between Thai and English. Language preference is preserved across sessions using localStorage.
